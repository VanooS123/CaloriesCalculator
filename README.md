# CaloriesCalculator
Программа сделана на Windows Forms. Сразу же при запуске открывается окно базовой формы, на которой можно посмотреть информацию об авторе,  справку (которая не сделана) и перейти в сам основной модуль программы.
На главной форме, по факту, вся программа. И тут есть 3 окошка: Норма калорий, Список блюд и Ингридиеты.
В Норме калорий можно посчитать свою норму калорий исходя из введенных показателей (стоит защита от неправильно введенных значений), а также можно рассчитать индекс массы тела и идеальный вес.
В "Ваши блюда" узнать о блюдах и их составе (чтобы узнать о составе, просто дважды нажимаем на всю строку).
В Ингридиентах перечислены все ингридиенты и можно создать новое блюдо.
